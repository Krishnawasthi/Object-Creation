# ☕ Java Object Creation

🚀 This repository contains Java programs that demonstrate the fundamentals of **Object Creation** in Java. It is designed for beginners who want to build a strong foundation in Core Java and Object-Oriented Programming (OOP).

Through simple and practical examples, this repository covers object creation using the `new` keyword, constructors, object references, instance variables, instance methods, and how objects interact within a Java program. Each example is written with readability in mind to help learners understand the concepts step by step.

## 📚 Topics Covered
- 🆕 Object Creation using the `new` keyword
- 🏗️ Default and Parameterized Constructors
- 🔄 Creating Multiple Objects
- 📦 Instance Variables
- ⚙️ Instance Methods
- 🔗 Object References
- 🧠 Stack and Heap Memory Basics
- 🎯 Accessing Object Members
- 💡 Real-world Java Examples

## 🛠️ Technologies Used
- ☕ Java
- 💻 Eclipse IDE
- 🌱 Git & GitHub

## 🎯 Goal
The main goal of this repository is to strengthen Core Java concepts and provide hands-on practice with object creation, making it easier to understand advanced OOP concepts like inheritance, polymorphism, encapsulation, and abstraction.

If you're learning Java from scratch or revising Core Java concepts, this repository will serve as a useful reference for understanding how objects are created and used in real-world applications.
